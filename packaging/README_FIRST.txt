=====================================
BabelAI 安装说明 / Installation Guide
=====================================

感谢下载 BabelAI！
Thank you for downloading BabelAI!

由于 BabelAI 是独立开发的应用，首次打开时 macOS 可能会显示安全警告。
这是正常现象，请按以下步骤安装：

Since BabelAI is independently developed, macOS may show a security warning 
when opening for the first time. This is normal. Please follow these steps:

-------------------------------------
安装步骤 / Installation Steps:
-------------------------------------

方法 1：右键打开（推荐 / Recommended）
1. 将 BabelAI 拖到 Applications 文件夹
2. 在 Applications 中找到 BabelAI
3. 右键点击（或按住 Control 键点击）BabelAI
4. 选择"打开"（Open）
5. 在弹出的对话框中，再次点击"打开"按钮

Method 1: Right-click to Open
1. Drag BabelAI to your Applications folder
2. Find BabelAI in Applications
3. Right-click (or Control-click) on BabelAI
4. Select "Open"
5. Click "Open" again in the dialog

-------------------------------------

方法 2：系统设置允许
1. 双击打开 BabelAI
2. 如果显示"无法打开"，点击"好"
3. 打开"系统偏好设置" → "隐私与安全性"
4. 在"安全性"部分找到关于 BabelAI 的提示
5. 点击"仍要打开"

Method 2: Allow in System Settings
1. Double-click BabelAI
2. If it shows "cannot be opened", click "OK"
3. Open System Preferences → Privacy & Security
4. Find the message about BabelAI in Security section
5. Click "Open Anyway"

-------------------------------------

方法 3：终端命令（高级用户）
打开终端，运行以下命令：
xattr -cr /Applications/BabelAI.app

Method 3: Terminal Command (Advanced)
Open Terminal and run:
xattr -cr /Applications/BabelAI.app
